Пример комманд:

1) переход по директориям

    cd <dir>

    cd ..

2) вывод дерева ФС

    ls .

    ls bin

    ls bin -p 2  //вывод с пейджингом

3) копирование файлов и папок

    copy File001 Dir01

    copy Dir01 Dir02

    copy File001 File002

4) вывод информации

    info File001

    info Dir01

5) удаление файлов и папок

    delete File001

    delete Dir01


6) Выход

    exit
